## Important files
```jax_cleanup.py```: the main file to run the code

```utils.py```: contains all parameters for the code

```algos/train_jax.py```: the training code

```evaluate.py```: the evaluation code

```algos/mappo_ippo_basic.py```: the network definitions

```socialjax/environments/cleanup/clean_up.py```: the env (and reward) definition

```requirements.txt```: to install the conda env (created from wsl)

```requirements.yml```: alternate to install the conda env (created from wsl)
